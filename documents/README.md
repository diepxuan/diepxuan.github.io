# Tài liệu nội bộ

Thư mục `documents/` chứa tài liệu nội bộ được xuất bản hoặc lưu trữ cùng website `docs.diepxuan.com`.

## Cấu trúc hiện tại

| Đường dẫn | Mục đích |
|---|---|
| `index.md` | Trang Jekyll hiển thị danh sách tài liệu tại `/documents/`. |
| `README.md` | Ghi chú vận hành thư mục tài liệu. |
| `README_DOCUMENTS.md` | Tài liệu mô tả chi tiết file hiện có. |
| `LEGISLATION_TRACKING.md` | Backlog theo dõi văn bản pháp luật mới. |
| `TonKho/` | Báo cáo tồn kho. |
| `WEBHD_INTERNET_UM_v1.0.docx` | Hướng dẫn dịch vụ Internet. |

## Quy tắc cập nhật

1. Chỉ đưa tài liệu đã được phép lưu trong repo.
2. Không commit tài liệu chứa mật khẩu, token hoặc dữ liệu nhạy cảm chưa được xử lý.
3. Nếu tài liệu cần hiển thị trên website, cập nhật `documents/index.md` trong cùng PR.
4. Nếu tài liệu là file nhị phân, mô tả mục đích và ngày cập nhật trong README hoặc trang index.
